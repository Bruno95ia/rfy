# Pastas por cliente

Para cada organização, crie uma subpasta com **slug** ou identificador estável (ex.: `acme-corp`) e **copie** os arquivos `01`–`06` da pasta pai para dentro dela, preenchendo com os dados daquele cliente.

Mantenha **segredos** (API keys, tokens) fora do Git; use apenas referências nos documentos `03`.
